# Spotify-display-2-
project on a spotify display, this was modeled in fusion360. Following the guide from hackclub, included a volume control rotary encoder and 3 keys for various api commands (examples: previous song, pause/play, next song). this model referenced 3d models from various authors on grabcad, sources below.
